## Instructions

`python >= 3.6` and `pipenv` are needed

0. Install requirements

```
pipenv shell && pipenv sync
```

1. Get data

```
./bin/get_data.sh
```

Now, in `data/<LANGUAGE>` you will have the datasets for each language.
